# Environment for Outputs Tables

This environment is created to store the outputs generated by a Redatam
program execution. The environment contains TABLES (Redatam outputs)
generated from FREQ, AREALIST, MATOP, TABLIST commands. You can't modify
objects contained in this environment.

## Usage

``` r
outputs
```

## Format

An object of class `environment` of length 0.

## Examples

``` r
if (FALSE) { # \dontrun{
dic<-redatam_open("path/to/rxdb")
df<-redatam_query(dic,"tbl1<-freq person.sexo")
# Accessing the tbl1 object from environment
redatamx::outputs$tbl1
# Accessing the output from Global Environment
print(df)
} # }
```
