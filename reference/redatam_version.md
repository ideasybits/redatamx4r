# Get the Redatam API version

Returns the current version of the native Redatam API.

## Usage

``` r
redatam_version()
```

## Value

String with the Redatam Engine version.

## Author

Jaime Salvador

## Examples

``` r
redatam_version()
#> [1] "RedEngine 1.2.1-final - Linux, compiled by 'GNU C++ version 13.3.0' on 'Nov 19 2025'"
```
