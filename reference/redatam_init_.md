# Initialize engine

Initialize the Redatam engine. This function is automatically called
when the package is loaded.

## Usage

``` r
redatam_init_(pachageDir)
```

## Arguments

- pachageDir:

  Relative path to Redatam runtime (dll, so or dynlib)

## Value

No return value.

## Author

Jaime Salvador

## Examples

``` r
if (FALSE) { # \dontrun{
redatam_init_("/engine");
} # }
```
